POGIL Activity Sets for IntroCS

Here are 5 collections of activities - 2 sets in Java, 3 in Python. Note that the first four sets of activities require you to request access to the full activities (with instructor guides and sample answers), by emailing contact@introcspogil.org with your preferred Google Drive account and which activity set(s) you are requesting access to.

  • ● Computer Science 1 in Java - 30+ activities by Helen Hu, assumes students have access to Java IDE
  • ● Computer Science 1 in Java - 20 activities by Chris Mayfield, can be used in classroom without Java IDE
  • ● Computer Science 1 in Python - 20 activities by Tricia Shepherd, Chris Mayfield, and Helen Hu
  • ● Computer Science 1 in Python - 10 activities by Clif Kussmaul
  • ● Computer Science 1 in Python - 20 activities by Lisa Olivieri
